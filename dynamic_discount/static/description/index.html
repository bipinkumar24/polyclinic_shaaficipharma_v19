Dynamic Discount

With the help of this Dynamic Discount module, users can set discount in ODOO, which effectively communicates the idea of flexible, responsive pricing schemes that are subject to change, depending on certain circumstances or situations.

Description

Description

Discover the power of our specially designed Dynamic Discount module, created to satisfy your specific needs. It can be applied to the total amount of the order as set in configuration. After installing this customized module, applied discount amount and discount type will print on invoice report.

Standout Features

Standout Features

  • Available on Community and Enterprise Editions on Odoo 14,15 and 16 versions.
  • Effective and easy say bye to observing every transaction to see whether the applied discount is appropriate.
  • Setting up of discounts through this module the discount can be easily allocated to the total bill.
  • Default Discount can be configure in the settings to apply the discount to the products.

Exploring the module: (How to)

Exploring the module: (How to)

  • Download and Install the module.
  • Go to POS > Configuration at home page and select the point of sale to begin session
  • Click on Shop to Configure Discount
  • Click on the desired discount % in Pricing to set the desired discount limit and save the settings.
  • Go to Dashboard and Start the session after the configuration.
  • Select the products to sell. Click on the Default Discount Button.
  • Select the Global Discount Type to be applied. (Amount)
  • We can set the discount applied separately on the Total Amount of Order.
  • We can select the Global Discount Type to be applied. (as a percentage)
  • Discount configured in the default settings will be automatically applied
  • Odoo automatically converts the percentage of discount allocated to order in amount, taking in regard the tax implications.
  • Click on the payment to continue further procedure. After choosing the payment type and customer, Validate the order. The receipt is generated, taking into account the Dynamic Discount.
