Direct Print

Print documents and product labels directly from your browser — no extra software needed.

Key Features

  • Browser Print Dialog — Opens your operating system's native print dialog so you can select any local, network, or virtual printer (PDF, etc.).
  • Product Label Printing — Print product labels in multiple formats (Dymo, 2×7, 4×7, 4×12) with or without prices, directly from products, sales orders, purchase orders, invoices, and stock transfers.
  • Direct Print for Any Report — Enable the "Use Direct Print" toggle on any Odoo report to make it open the browser print dialog instead of downloading a PDF.
  • Gear Menu Integration — "Direct Print" appears in the action/gear menu on Contacts, Sales Orders, Purchase Orders, Invoices, and Stock Transfers.
  • Header Buttons — Quick-access print buttons in the form header of key documents.
  • No External Dependencies — Works entirely in the browser. No printer agents, browser extensions, or Java applets required.

How It Works

  1. Install the module.
  2. Open any supported document (e.g. a Sales Order).
  3. Click Direct Print in the header or gear menu, or Print Labels on a product form.
  4. Your browser's print dialog opens — select your printer and print!

Configuration

To make any Odoo report use Direct Print by default:

  1. Go to Settings → Technical → Reports.
  2. Open the report you want to modify.
  3. Enable the Use Direct Print checkbox.

From now on, clicking that report will open the browser print dialog instead of downloading a PDF.
