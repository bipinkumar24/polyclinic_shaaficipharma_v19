POS Load Product Location module displays only those products which belong to the location configured in the POS setting. In standard Odoo all the products are loaded irrespective of the location.

Key Features

Shows products belonging only to the location set for a particular POS.

Results in faster search, faster loading of the POS.

Improves overall operating efficiency.

Enable Multiple Stock location

  • Navigate to the Settings menu >> Users & Companies >> Users.
  • Now open Access Rights tab >> Technical Settings section.
  • Enable Manage Multiple Stock Locations to access warehouse location.
Open Location Configuration

  • Navigate to Inventory >> Configuration >> Locations.
  • Choose Location.
WH/Stock/Shelf 1

Open Current stock

  • Some Products are available in this stock.
Open POS

  • Navigate to Settings in POS Shop.
POS Shop Settings

  • Select Stock Location in Inventory Tab.
  • Selected Stock location's product loaded in POS.
Open POS Session

  • Only WH/Stock/Shelf 1 product are to be shown in POS.
